# Security Policy

This repository is a **demonstration / portfolio project** intended for learning
and evaluation purposes.

## Reporting a vulnerability

If a security issue is discovered, please open a GitHub issue including:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if available)

## Supported versions

Dependencies should be kept up to date via:
- Dependabot
- Regular Maven updates

This project is not intended for production use.
